OrderRx

Summary

This was built for a pharmacy to replace their current method of ordering prescriptions on behalf of patients, which was done with just an excell document. My solution was to give them an intuative ui that would show them each day what requests need to be made. The system would then just email the patient's GP with the medication request.

Tech Stack

  • React
  • C# & .NET
  • Shadcn
  • JWT Authentication
