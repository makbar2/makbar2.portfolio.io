PharmaFriends

Overview

I took over the development of a full-stack web application that enabled pharmacists to carry out New Medicine Service (NMS) consultations remotely. The application was originally built using Node.js, Express, and a SQL database. I restructured the backend to make it more modular, added new features to both the front and backend, and designed and developed the database ensuring schema was in third normal form to reduce data redundancy. I also built tools within the web-app for non-technical users to audit data without directly accessing the database. JWT authentication was used to protect routes and manage access to sensitive data.

Technologies used

  • Bootstrap
  • Node.js
  • Express
  • SQL
