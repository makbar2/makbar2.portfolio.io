PharmaFriends

Overview

I took over development of a full-stack web application enabling pharmacists to perform New Medicine Service (NMS) consultations remotely. The app was originally built with Node.js, Express, and a SQL database. I restructured the backend by modularizing which helped with maintainabiltiy and scalabiltiy , improved the database design (ensuring 3NF to reduce redundancy) ,and added features across both front and backend.

The application also featured:

  • Jwt authentication to protect routes
  • Audit tools to allow non-technical people to audit data without having to access the database directly.
